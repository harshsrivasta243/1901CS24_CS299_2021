python_code.py is the front-end code for the web app using streamlit, back-end is the python code for model training and testing along with designating classifier.pkl based on
logistic regression while the self-generated dataset is in both formats .txt and .csv.
To run the web app, go to your anaconda prompt, change directory to the one in which the files are saved, and then command "streamlit run python_code.py" to let the prompt display
the URL of the web app and also open it in your default browser. Feed the numeric details and the predictor would tell you whether there is a higher chance of loan amount approval
or dismissal.
